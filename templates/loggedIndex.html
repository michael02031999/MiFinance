    How to use MiFinance

    Data

    Upon login, the user will be presented with a pie chart and line chart. The pie chart has no real values yet because the user did not input any, and the same is true of the line chart. The pie chart shows distribution of how you spend your money for the current month. The line chart shows your monthly spendings. At the very top, you have the ability to input cash. The cash amount does not affect the data. Essentially, this number should mirror your bank account!

    Journal

    The Journal section is where you will be keeping track of every transaction. It is important to note that to write a successful journal entry, you must input the month number, transaction type, transaction name, and price. When you click submit, the transaction should be displayed below. When you click to see the data, data will only be shown for the current month.
